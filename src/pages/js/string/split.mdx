## js

```js
const title = "Hello PlayTip!";
// 공백으로 나눔, (2) ['Hello','PlayTip']
console.log(title.split(" "));

const title = "playtip";
// (7) ['p', 'l', 'a', 'y', 't', 'i', 'p']
console.log(title.split(""));
```

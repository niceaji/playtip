## css

```css
div {
  margin: 10px 0;
}
.underline {
  text-decoration: underline;
}
.overline {
  text-decoration: overline;
}
.through {
  text-decoration: line-through;
}
.dotted {
  text-decoration: underline dotted red;
}
.wavy {
  text-decoration: underline wavy green;
}
```
